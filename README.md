# 📊 E-Commerce Analytics Dashboard

## 📌 Overview
Interactive Power BI dashboard analyzing
100K+ real orders from Olist Brazil
E-Commerce platform (2016–2018)

## 🎯 Key Metrics
| Metric | Value |
|--------|-------|
| Total Revenue | $16M |
| Total Orders | 99K |
| Total Customers | 96K |
| Avg Rating | 4.1 ⭐ |

## 📊 Dashboard Features
- ✅ 5 KPI Cards with YoY Growth
- ✅ Revenue Trend Line Chart
- ✅ Orders by Status Donut Chart
- ✅ Payment Type Analysis
- ✅ Rating Distribution
- ✅ Top States by Revenue
- ✅ Interactive Slicers

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX
- Data Modeling

## 📁 Dataset
- Source: Kaggle
- Name: Olist Brazilian E-Commerce
- Records: 100K+ orders
- Period: 2016–2018
